# Dynamic File Compression Utility

A Python-based file compression and decompression utility that uses the Huffman Coding Algorithm to reduce file size efficiently while preserving data integrity.

## Features

✅ Huffman Coding Compression

✅ File Decompression

✅ Compression Ratio Calculation

✅ Dynamic Huffman Code Generation

✅ Lossless Data Recovery

✅ Simple Command Line Interface

## Technologies Used

- Python
- Data Structures
- Huffman Tree
- Priority Queue (Heap)
